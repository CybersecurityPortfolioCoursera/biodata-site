# 🌐 Biodata Website - Shaurya Tiwari

This is a modern, responsive biodata website created using HTML, CSS, and JavaScript. It features a futuristic glassmorphism UI with smooth animations and a clean layout.

## 🚀 Features
- Anti-gravity floating glass UI design
- Fully responsive (mobile + desktop)
- Smooth scrolling navigation
- Animated sections with reveal effects
- Resume download functionality
- Modern dark theme with gradient background
